# 🎓 Flashcard Master Pro

An interactive and responsive Flashcard Learning Application developed using **HTML, CSS, and JavaScript**. This project helps users learn technical concepts through digital flashcards with features such as voice support, search, favorites, and progress tracking.

---

## 🚀 Features

* 📚 Interactive Flashcards
* 🔄 Next and Previous Navigation
* 👁️ Show/Hide Answer
* 🔊 Text-to-Speech (Question + Answer)
* ➕ Add New Flashcards
* ✏️ Edit Existing Flashcards
* ❌ Delete Flashcards
* 🔍 Search Flashcards
* ⭐ Favorite Flashcards
* 📊 Progress Tracking
* ⏱️ Timer
* 🌙 Dark Mode
* 🔀 Shuffle Cards
* 📱 Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Run

1. Download or clone the repository:

   ```bash
   git clone https://github.com/your-username/flashcard-master-pro.git
   ```

2. Open the project folder.

3. Run `flashcard.html` in your browser.

---

## 💡 Application Features

* Users can browse flashcards using navigation buttons.
* Questions and answers can be read aloud using the Speech API.
* New flashcards can be created dynamically.
* Existing flashcards can be edited or deleted.
* Search functionality helps users quickly find flashcards.
* Progress and learning statistics are displayed in real time.
